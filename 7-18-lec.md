# 7-18 Lec

Operations: Concat, filter, slice, splice, sort, copyWithin,

```javascript
arr.sort(); // Default sort order is according to string Unicode code points
arr.sort(fn); // customize sort order by defined function
// e.g.
arr.sort((a, b) => a-b); 
// Equivalent to
function comp(a, b) {
    return a-b;
}
arr.sort(comp);
```
